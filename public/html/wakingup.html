Excellent! Let's work with comparators and boolean operators. The common number comparators in JavaScript are:

  • == which returns true when the left side and right side are equal. Else, it returns false
  • != which returns true when the left side and right side are not equal. Else, it returns false
  • > which returns true when the left side is greater than the right. Else, it returns false
  • < which returns true when the left side is less than the right. Else it returns false
  • => which returns true when the left side is greater than or equal to the right. Else, it returns false
  • =< which returns true when the left side is less than or equal to the right. Else it returns false

So what will the following code return, Detective?

console.log(6 > 4); //C

Good job, Detective! When it comes to strings, the plus sign concatenates them. For example:

console.log("Raindrops On Roses" + " And " + "Whiskers On Kittens");

Would return:

"Raindrops On Roses And Whiskers On Kittens"

You can concatenate numbers into your string, too! But keep in mind of spaces. Expressions get evaluated.

console.log("Platform" + 9 + "and " + 3/4);

Would return:

"Platform9and 0.75"

While:

console.log("Platform " + 9 + " and 3/4");

Would return:

"Platform 9 and 3/4"
